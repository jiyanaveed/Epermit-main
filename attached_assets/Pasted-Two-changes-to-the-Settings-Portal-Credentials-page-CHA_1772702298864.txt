Implementation details:

1. The Jurisdiction field should be a searchable combobox/autocomplete. 
   User can type to filter the list. They can also type a custom 
   jurisdiction name not in the list (for new jurisdictions).

2. When a jurisdiction is selected from the dropdown:
   - Auto-fill the Portal URL field with the matching URL
   - If the URL is empty string, leave Portal URL empty for manual entry
   - User can always manually edit the Portal URL after auto-fill

3. When editing an existing credential, pre-populate the Jurisdiction 
   dropdown with the saved jurisdiction name and show the saved URL.

4. The Portal URL field should remain editable even after auto-fill, 
   so users can correct or update URLs.

5. If user types a jurisdiction not in the list, Portal URL stays 
   empty for manual entry.

Do NOT change any backend logic, API calls, or database schema.
Do NOT change any other pages or components.
Only update the credential form UI in Settings.
